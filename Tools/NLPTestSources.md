# NLP Test Sources (Paraphrased)

We used the sources below for **lightly paraphrased** natural-language task phrases.  
No phrases are copied verbatim; structures are adapted for student-heavy daily planning.

## Sources

- [Calendars by Readdle — Natural Language Input examples](https://support.readdle.com/calendars/tips-and-tricks/natural-language-input)  
  Patterns paraphrased: 24h time (“meeting at 23:45”), spelled-out hour (“meeting at four”), “lunch at noon for 30 minutes”, “meeting with Tom”, “event in 5 days”, “meeting at 7 pm”.

- [BusyCal iOS — URL Handler examples](https://www.busymac.com/docs/busycalios/139998-url-handler/)  
  Patterns paraphrased: “project review 3pm”, “baseball game tomorrow”, task phrasing like “call Bob tomorrow”.

- [BusyCal (macOS) — URL Handler examples](https://www.busymac.com/docs/busycal/70621-url-handler/)  
  Patterns paraphrased: “staff meeting Thursday at 10am”, “meeting with Joe June 7 at 3pm /Work”.

- [Busy Apps blog — BusyCal URL Handler](https://support.busymac.com/dev/blog/23829)  
  Patterns paraphrased: “meeting with Joe June 7 at 3pm /Work”, “conference call Tuesday at 2pm”.
